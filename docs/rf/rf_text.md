# 射频测试页面

在高速产品中物理设计和机械设计都会导致信号完整性的问题。

信号完整性指的是在高速产品中由于互连线引起的所有问题，它主要研究当互连线与数字信号的电压、电流波形相互作用时，其电器特性如何影响产品性能。

这些问题可以大致归类为以下三类：
1. 信号完整性（Signal Integrity，SI）信号波型的失真
2. 电源完整性（Power Integrity，PI）有源器件供电的互连线及各相关元件上的噪声
3. 电磁兼容（ElectorMagnetic Compatibility，EMC）产品自身产生的电磁辐射和由外场导入产品的电磁干扰

电磁兼容部分其一是产品自身产生了大量的电磁辐射进入外界；其二是源于外界辐射严重干扰了产品。

信号完整性噪声问题有很多诱因比如：振铃、反射、近端串扰等，看起来这些互连线的电气特性对数字信号波形造成了不同程度的影响，这些术语和设计过程中的需要避开的坑看起来非常多让人无暇顾及到所有，但是这类诱因都与信号完整性/电源完整性/电磁兼容领域中下述 6 种类型的问题之一有关：
1. 单一网络的信号失真
2. 互连线中频率相关损耗引起的上升边退化
3. 两个或多个网络之间的串扰
4. 作为串扰特殊形式的地弹和电源弹
5. 电源和地分配中的轨道塌陷
6. 来自整个系统的电磁干扰和辐射

一旦知道了与 6 总问题相关的根源，找出和解决这类问题的一般方案就显而易见了，这些问题在所有的互连线种都起作用，小到芯片之间的连接线大到板级电缆连接线等任何位置之间的互连线，其原理和效果是一样的。

## 单一网络的信号完整性

某一个网络由系统中所有连接在一起的金属组成，其中一个网络不仅包括信号路径，还包括信号的返回路径。

反射：引起反射的唯一原因就是信号遇到的瞬时阻抗发生改变，信号感受到的瞬时阻抗与信号路径和返回路径的物理特性有很大的关系。
当信号从信号驱动器输出的时候构成信号的电流和电压将互连线看成一个阻抗网络，当信号沿着网络传播时它不断感受到互连线引起的瞬时阻抗变化，如何互连线的阻抗保持均匀不变则信号不失真。然而一旦阻抗发生变化，信号就会在变化处产生反射并通过互连线的剩余部分时继续失真。如果这个阻抗变化足够大则可能会导致系统误触发。

其中任何改变互连线横截面或者网络几何特征都会改变信号感受到的阻抗，引起阻抗变化的所有特征成为**突变**导致信号所感受到的阻抗发生改变。

导致阻抗发生变化的情况可能来自以下几点：
1. 互连线末端，通常是接收器的开路高输入阻抗或者驱动器的低输出阻抗，如果反射噪声的源头是瞬时阻抗的变化，那么解决这个问题的办法就是将互连线的阻抗设计为恒定的。
    1.1 使用线条阻抗为常量或者所谓“可控”的电路板，通常意味着均匀的传输线
    1.2 为了控制末端的反射，采用电阻的端接匹配策略去控制反射，让信号看不到有阻抗的变化
    1.3 使用沿线拓扑的阻抗维持恒定的布线规则，这就要采用点到点布线，最小化支路长度或短桩线
    1.4 设计不均匀的传输线结构，以减轻线的不连续性，这需要对线的几何特征进行精确设计以修正边缘场
2. 线宽变化
3. 层转换
4. 返回路径平面上的间隙
5. 接插件
6. 路由拓扑的改变，比如分支线、T 型线或者桩线

单一网络中还存在第二个方面的信号质量问题，这就是导线和介质中与频率相关的损耗所造成信号的高频损耗比低频损耗更大，其结果是在传播过程中信号上升边沿会被拉长，当这个上升边沿退化到接近信号的单位间隔时，此时 1 bit 的信息会泄露到下一个甚至下下个比特，这种效应称为符号间干扰。

单一网络中影响信号质量的第三个方面就是时序，两个或者多个信号路径之间的时延差成为错位，当信号线和时钟线之间存在超出预期的错位时就可能会产生误触发和逻辑错误

错位是一个时序问题，多数是由互连线的电气特性引起的，互连线的总长度对错位影响最大，只要在版图设计的时候自己匹配互连线之间的长度就能较为轻易地解决问题。

## 串扰

当网络传播信号的时候有些电压和电流能传递到临近的静态网络上，如果这个网络正在忙自己的事情时当前网络的信号就会以有害噪声的形式耦合到第二个静态网络上。
